# MacroTrack 🥗📊

A modern macro and nutrition tracking web application built with **Next.js**, **TypeScript**, and **Tailwind CSS**.

This project was created as a hands-on learning exercise to explore modern AI-assisted software development using tools such as **Cursor**, **Claude**, **Vercel**, and the Node.js ecosystem.

---

## 🚀 Project Overview

MacroTrack helps users track their daily nutrition intake by recording meals and monitoring key macronutrients:

- Calories
- Protein
- Carbohydrates
- Fat

Users can add meals throughout the day and instantly see their progress against daily nutrition goals.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/macrotrack.git
cd macrotrack
```

Install dependencies:

```bash
npm install
```

Create environment variables:

```bash
cp .env.example .env.local
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
